# 90 Days

Gamified self-improvement PWA for iPhone. Level up your pixel character as you complete daily tasks and build habits over 90 days.

## Getting Started

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
npm start
```

## Deploy to Vercel

1. Push this repository to GitHub
2. Go to [vercel.com](https://vercel.com) and import the repository
3. Click Deploy (zero configuration needed)
4. Open on iPhone via Safari → Share → Add to Home Screen

## Tech Stack

- **Next.js 15** — App Router, SSR/SSG
- **TypeScript** — Full type safety
- **TailwindCSS** — Utility-first styling
- **Framer Motion** — Smooth animations
- **Zustand** — Global state management
- **next-pwa** — Service worker & PWA support
- **LocalStorage** — Persistent client-side storage
- **date-fns** — Date utilities

## Features

- 🎮 Animated pixel character (4 tiers: Beginner → Discipline → Momentum → Transformation)
- ⭐ XP system with 50 levels
- ✅ Daily task tracking with 8 categories and 3 priority levels
- 📅 90-day challenge calendar
- 📊 Analytics with 7-day XP chart
- 📝 Notes with inline editing
- 🔥 Streak tracking
- 🌙 Dark mode + automatic night atmosphere (10pm–6am)
- 📱 PWA — installable on iPhone via Safari
- 💾 All data stored locally (no backend needed)
